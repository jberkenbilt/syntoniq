# Highlights

This release includes the first round of fixes as I actually use this for real work. There is more granularity for setting start and end position for generation, and boundaries are correct for in-flight changes to tempo, pitch, and dynamics. There are also enhancements to `syntoniq-kbd prompt` and more diagnostic features.

# Highlights

This release adds the `syntoniq-kbd prompt` command, an interactive command-line tool for building chords a note at a time. There is one breaking change: a bug fix to the interpretation of generated notes with `!n` overrides. With just `!n`, the note is aligned with _n_ divisions of the *octave*, not of the scale's default interval as in prior versions. This way, notes with `!` override have the same value in all tunings, which is the intended behavior.
